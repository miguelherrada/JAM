JAM: Numerical Methods for Curved Spaces

Welcome to the official page for JAM, a numerical framework designed for solving equations in curved spaces.

Features

  • Independent of the discretization scheme
  • Optimized for computational physics applications

Installation

        cd JAM
        python setup.py install
